# Interview Data Package

This package contains ready-to-save interview data prepared for repository use.

## Structure

- `raw/`: near-verbatim interview transcripts in plain text
- `summaries/`: bullet-point summaries for each interview
- `json/`: structured JSON files for each interview
- `README.md`: package description

## Current Sample

- 4 Mainland Chinese student interviews
- 2 International student interviews
- Total: 6 interviews

## File Naming

- `mainland_01` to `mainland_04`
- `international_05` to `international_06`

## Suggested Git Commands

```bash
git init
git add .
git commit -m "Add raw transcripts, summaries, and JSON interview data"
git branch -M main
git remote add origin <your-repo-url>
git push -u origin main
```
