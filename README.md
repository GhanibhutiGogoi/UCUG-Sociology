# UCUG Sociology

This repository is for the **UCUG Sociology** class.

## About

UCUG Sociology is an introductory sociology course that explores the fundamental concepts of human society, culture, and social behavior. The course covers topics such as:

- Social structures and institutions
- Culture and socialization
- Social stratification and inequality
- Race, ethnicity, and gender
- Deviance and social control
- Social change and globalization

## Course Materials

This repository contains notes, assignments, and resources related to the UCUG Sociology class.
